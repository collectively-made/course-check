# course-check

> Electron + AngularJS with Grunt workflow.


## Getting Started

```
$ npm install && bower install
```

### Run

```
$ npm start
```

### Test

```
$ grunt test
```

Builds the app for OS X, Linux, and Windows, using [electron-packager](https://github.com/maxogden/electron-packager).
